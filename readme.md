# 结论

总体来说YOLO不论是分类还是目标检测，基本上做到了速度和精度的均衡。

openvino加速可以比CPU推理快6倍左右，但需要CPU是英特尔平台并且有集成显卡。精度有一定程度下降，平均下降2-3%。onnx推理精度几乎保持不变，速度提升约3倍。

tensorrt加速可以比GPU推理快3倍左右，需要GPU为英伟达平台。精度基本保持不变，下降在1%内。
